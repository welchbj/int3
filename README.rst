Synopsis
--------

`int3` is a Python library and command-line tool for writing and working with position-independent assembly:

```sh
echo -n "int3" | int3 assemble | int3 format
b"\xcc"
```

Installation
------------

int3 is tested on the latest three major versions of CPython. You can get the latest release from PyPI with::

    pip install int3
